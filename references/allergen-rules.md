# 台灣食品過敏原標示規則

依據《食品過敏原標示規定》（衛授食字第1021350146號）及後續修正，
以下原料及其製品須強制標示。

## 必標過敏原（共11類）

| 過敏原類別 | 常見原料關鍵字 |
|-----------|--------------|
| 蝦及其製品 | 蝦、蝦仁、蝦米、蝦皮、蝦醬 |
| 蟹及其製品 | 蟹、螃蟹、蟹肉、蟹膏 |
| 貝類及其製品 | 牡蠣、蛤蜊、扇貝、干貝、鮑魚、蚌、蜆 |
| 魚類及其製品 | 魚、鯖魚、鮪魚、鮭魚、柴魚、魚露、魚板、魚漿、烏魚、吳郭魚 |
| 蛋類及其製品 | 蛋、雞蛋、鴨蛋、蛋白、蛋黃、卵磷脂（蛋源）、美乃滋 |
| 花生及其製品 | 花生、花生醬、花生油 |
| 小麥及其製品 | 小麥、麵粉、麵筋、麩質、醬油（含小麥釀造者）、烏醋、麵包粉 |
| 大豆及其製品 | 大豆、黃豆、豆腐、豆漿、味噌、醬油（含大豆者）、大豆油（精製者可免標） |
| 芝麻及其製品 | 芝麻、麻油、芝麻醬 |
| 牛奶及其製品 | 牛奶、乳清、乳糖、奶粉、奶油、起司、鮮奶油、酪蛋白 |
| 堅果及其製品 | 杏仁、腰果、核桃、榛果、開心果、夏威夷豆、松子、栗子 |

## 自動偵測邏輯

比對成分表中所有原料名稱（含複合原料的子原料），
找到符合上表關鍵字者，列入過敏原警語。

### 輸出格式

```
本產品含有○○、○○及其製品。
```

若無任何過敏原：
```
本產品不含主要過敏原，但於同廠生產線作業，可能含有微量○○。
（視實際生產情況決定是否加此備註）
```

## 常見判斷案例

- **醬油**：通常含大豆＋小麥 → 標示「大豆、小麥」
- **美乃滋**：含蛋、大豆 → 標示「蛋類、大豆」
- **豆瓣醬**：含大豆 → 標示「大豆」
- **柴魚粉**：含魚類 → 標示「魚類」
- **乳清蛋白**：含牛奶 → 標示「牛奶」
- **卵磷脂（大豆）**：含大豆（精製大豆卵磷脂可視情況）
