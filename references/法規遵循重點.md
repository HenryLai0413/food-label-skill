# 食品標示法規遵循重點（強制）

> 依據《食品標示法規手冊(中文版)》《食品標示法規指引手冊》《包裝食品營養標示應遵行事項》
> （111.6.23 衛授食字第1111301261號修正）《食品過敏原標示規定》（107.8.21）。
> 本檔規則**優先於** SKILL.md 內其他寫法；產出標示前必須逐項核對。

---

## 1. 營養標示｜熱量必用 Atwater 計算（最常錯）

熱量**不可**用各食材熱量直接加總，必須：

```
熱量(大卡) = 蛋白質(g)×4 + 脂肪(g)×9 + 碳水化合物(g)×4
```
（膳食纖維×2；糖醇×2.4，赤藻糖醇×0；有機酸另計）

- 審查會核對「宣稱的三大營養素能否還原出宣稱熱量」，用食材熱量加總會對不起來。
- 三大營養素本身可用**配方計算法**（手冊Q7.2）：列配方百分比→查營養資料→計算→**考慮水分變化**→**考慮加工損耗與保留率**。保留率/守恆算法是法規認可的。

## 2. 營養標示｜標準格式與修約

**標準格式（固定採用此兩欄式，Henry 指定）：**
```
              營養標示
每一份量　X公克
本包裝含　Y份
                     每份            每100公克
熱量                 ○○大卡          ○○大卡
蛋白質               ○○公克          ○○公克
脂肪                 ○○公克          ○○公克
　飽和脂肪           ○○公克          ○○公克
　反式脂肪           ○○公克          ○○公克
碳水化合物           ○○公克          ○○公克
　糖                 ○○公克          ○○公克
鈉                   ○○毫克          ○○毫克
```
- 飽和脂肪、反式脂肪縮排於脂肪下；糖縮排於碳水化合物下。
- 若每份量＝100公克，兩欄數值相同亦照列。
- **無數值一律填「0」**（不留空；0 亦如實顯示）。
- **數值與單位靠右對齊**（單位皆兩字，靠右即對齊）。
- **只列 8 項強制營養素；宣稱/自願標示之營養素（鈣、維生素C、精胺酸等）不列出**（除非該產品確有宣稱）。

- 欄位亦可採 **每份＋每日參考值百分比**（次選）。
- 順序固定：熱量、蛋白質、脂肪、（飽和脂肪、反式脂肪）、碳水化合物、（糖）、鈉，其後接宣稱營養素。
- 修約：四捨五入，或 CNS 2925「四捨六入五成雙」。**脂肪等以整數或小數點後一位為原則**。
- 「得以0標示」門檻（每份及每100g/100mL 皆須低於界限才可標0）：
  | 項目 | 標0門檻 |
  |---|---|
  | 熱量 | ≤4 大卡 |
  | 蛋白質 / 脂肪 / 碳水化合物 / 糖 | ≤0.5 公克 |
  | 飽和脂肪 | ≤0.1 公克 |
  | 反式脂肪 | ≤0.3 公克 |
  | 鈉 | ≤5 毫克 |
  - 判定用**實測/計算值**，不可先修約再判定（0.51→不可標0）。

## 3. 過敏原標示

**強制標示（11類）**：甲殼類、芒果、花生、牛奶(羊奶)、蛋、堅果類、芝麻、含麩質之穀物、大豆、魚類、亞硫酸鹽類（終產品二氧化硫殘留≥10mg/kg）。

**建議標示（非強制）**：頭足類、**螺貝類（含牡蠣/蠔）**、種子類、奇異果。
→ **立康標示原則（Henry 指定）：過敏原警語只列強制 11 類，建議標示項一律不列入。**

- 措辭（Henry 指定固定格式）：「**本產品含有……。**」——**不寫「過敏原」「的過敏原」**，也不加「不適合其過敏體質者食用」。麥類寫「麥類、含麩質的穀物」。例：「本產品含有大豆、麥類、含麩質的穀物。」
- 交叉污染（共線）用建議語：「本產品生產製程廠房，其設備或生產管線有處理○○」。
- 複合原料帶入的致敏成分（如醬油含小麥、豆瓣含芝麻）仍須揭露。

## 4. 內容物（成分）名稱與排序

- 二種以上混合物：**依成品含量多寡由高至低**排列（非投料量；嫩化醃漬助劑多數瀝除，成品含量小應靠後）。
- 添加且殘留於成品的**水**須列入（依含量排序）。
- 複合原料須以括號展開子成分（含其中的添加物、致敏原）。
- **肉品主原料命名（Henry 指定）：只寫「雞肉」「豬肉」等，不寫部位、不加去皮去骨等描述**（營養計算基準另記於備註）。
- **成分表若有豬肉或牛肉，其後須以括號標示原產地（Henry 指定）**：如「豬肉（台灣）」「牛肉（澳洲）」。此時廠商資訊區的「原產地」可只寫整體產地、不需再單列豬/牛原產地。

## 5. 食品添加物標示（食安法第22條第1項第4款）

- 添加物名稱**須使用中央主管機關所定之品名或通用名稱**，**不得僅以功能（用途）名稱標示**。
- **只有**甜味劑、防腐劑、抗氧化劑三類：**須同時標示功能性名稱＋品名/通用名稱**（例：「防腐劑（己二烯酸鉀）」）。
- **其他所有添加物：只標品名/通用名稱即可，不需（也不必）加功能性名稱**。
  - 例：碳酸鈉 → 標「碳酸鈉」（不寫「酸鹼調整劑（碳酸鈉）」）；DL-胺基丙酸 → 標「DL-胺基丙酸」；味精 → 標「L-麩胺酸鈉」。
- 混合二種以上添加物**以功能性命名者，應分別標明各添加物名稱**（複方須展開，例：綜合磷酸鹽 → 多磷酸鈉、多磷酸鉀、焦磷酸二氫二鈉、焦磷酸鉀）。
- **「修飾澱粉」是類別名，不可單獨標示**，須寫個別品名/學名（如乙醯化己二酸二澱粉、羥丙基磷酸二澱粉、醋酸澱粉、磷酸二澱粉、氧化澱粉…）。依供應商規格圖確認實際品名。
- 帶入原則：經合法原料帶入、含量明顯低於需用量且對終產品無功能者，得免標示。

## 6. 內容物淨重／固形量

- 固、液兩相且固形物為主要價值者，須**分別標示內容量（淨重）與固形量**（如醬菜、罐頭）。
- 收汁類（醬汁不可分離）以淨重標示即可；若成品有可分離醬汁，應加標固形量（需實測）。
- 淨重容許負誤差依《定量包裝商品管理辦法》附表（非固定±5%），由 net_weight 查表。
- **顯示方式（Henry 指定）：誤差一律寫百分比、不寫公克**（例：淨重500公克、±3%，不寫「±15公克」）。generate_docx 已改為此。

## 7. 其他

- 有效日期：西元年/月/日；可打印於封口，標示位置註明。
- 品名：如實反映本質；非素食品名不加素食括號。
- 廠商：自有品牌標「製造廠商」；OEM 標「負責廠商」（客戶資料）。
