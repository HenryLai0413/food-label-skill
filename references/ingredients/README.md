# 商用原料規格資料夾

## 用途
存放商用原料（複合原料、加工食品）的成分表與營養標示，供 Claude 在生成食品標示時本地查詢，
取代估算，提高準確度。

## 檔案命名規則
```
{原料品名}_{品牌或供應商}.md
```
例：`雞粉_明德食品.md`、`辣椒醬_牛頭牌.md`

## 檔案格式範本

```markdown
# {原料品名}（{品牌}）

## 基本資訊
- 供應商：
- 品牌：
- 規格：（e.g. 1kg/包）
- 更新日期：

## 子成分（依含量排序）
{原料1}、{原料2}（子原料A、子原料B）、{原料3}…

## 過敏原
含有：（e.g. 大豆、小麥）

## 每100公克營養成分
| 項目 | 數值 |
|------|------|
| 熱量 | __ 大卡 |
| 蛋白質 | __ 公克 |
| 脂肪 | __ 公克 |
| 飽和脂肪 | __ 公克 |
| 反式脂肪 | __ 公克 |
| 碳水化合物 | __ 公克 |
| 糖 | __ 公克 |
| 鈉 | __ 毫克 |

## 備註
（e.g. 含防腐劑、建議添加量、保存方式）
```

## 使用說明
當配方中出現此資料夾內有紀錄的原料時，Claude 會：
1. 優先讀取此檔案的子成分展開（取代猜測）
2. 使用此檔案的每100g營養值計算加權貢獻
3. 自動帶入過敏原

## 資料來源建議
- 向供應商索取原料規格書（Spec Sheet）
- 掃描或輸入規格書上的成分表與營養標示
- 每次換供應商時更新對應檔案
