# 食品標示自動生成 Skill

依據《食品安全衛生管理法》第22條及相關子法，根據使用者提供的資料，
生成完整合規的食品標示，並產出 Word 文件（.docx）。

---

## 快速上手：如何使用這個 Skill

### 觸發方式

在對話中說出以下任何一句即可啟動：
- 「幫我做 ○○ 的食品標示」
- 「生成 ○○ 的標示草稿」
- 「寫 ○○ 的成分表」
- 「產出 Word 標示」

### 需要提供的資料

#### 必填項目（缺少時會逐一詢問）

| 項目 | 說明 | 範例 |
|------|------|------|
| **品名** | 純品名，不加素食/非素食括號 | `麻油猴頭菇湯` |
| **品牌或廠商類型** | 決定廠商資訊與素食標示方向 | `津沅寶` 或 `代工` |
| **配方原料** | 各原料的用量（公克）或百分比 | 見下方格式範例 |
| **淨重** | 成品標示重量（公克） | `500` |
| **保存方式** | 含溫度條件 | `常溫保存，開封後冷藏並盡速食用` |
| **份量設計** | 每份幾克、本包裝幾份 | 每份 250g，共 2 份 |

#### 選填項目（有則更準確）

| 項目 | 說明 |
|------|------|
| **固形物重量** | 湯品類需標示固形物含量（公克） |
| **營養成分數值** | 如有送驗報告可直接提供；沒有則系統查 TFND 估算 |
| **有效日期標示位置** | 如「標示於封口處」；省略則填「標示於包裝上」 |
| **原產地** | 預設台灣 |
| **備註** | 需加在標示底部的額外說明 |

#### 代工產品需額外提供

| 項目 |
|------|
| 委託廠商（客戶）公司名稱 |
| 客戶電話 |
| 客戶地址 |
| 工廠登記號碼（客戶的，或協議使用立康的 93773108） |

### 配方格式：三種寫法都接受

**公克數**
```
水 255g、猴頭菇（泡發後）200g、薑 15g、麻油 20g、食鹽 3g、白胡椒粉 1g
```

**百分比**
```
水 51%、猴頭菇 40%、薑 3%、麻油 4%、食鹽 0.6%、白胡椒粉 0.2%
```

**食譜式描述**
```
這鍋湯共 500g：猴頭菇乾貨 50g 泡發、水 255g、麻油 20g、薑 3 片（約 15g）、鹽少許
```

> 若原料是商用複合品（如「穀盛味霖」、「金蘭醬油」、「安安辣豆瓣醬」），直接寫品牌+品名，系統會自動去 Google Drive 圖庫抓規格書展開子成分。

### 系統自動處理的事項

- 成分依含量由多到少排序
- 商用原料展開子成分（查 `G:\我的雲端硬碟\立康-產品&原物料成本\原料成分&營養標` 圖庫）
- 過敏原自動偵測並產出警語
- 素食類型判定（並標註在品名後）
- 廠商資訊自動帶入（煨小寶 / 津沅寶 / 呷8味）
- 淨重容許誤差自動依法規附表計算
- 營養成分查 TFND 2025 資料庫估算（如無送驗值）
- 輸出 Word 文件（.docx，Noto Sans TC 字型，檔名自動命名）

### 最快的開始方式

直接說：
> 「幫我做**麻油猴頭菇湯**的食品標示，品牌**津沅寶**，淨重 **500g**，原料：水 255g、猴頭菇（泡發後）200g、薑 15g、麻油 20g、食鹽 3g」

剩下不足的資料系統會詢問，確認後產出 .docx 檔案。

## 輸入資料收集

使用者至少需提供：
- **品名**（純品名，**不加**素食/非素食括號，例如「燉雞湯」而非「燉雞湯（非素食）」）
- **配方或原料清單**（含各原料用量或百分比；若為複合原料需提供其子原料）
- **淨重**（需標明固形物比例，若適用）
- **保存方式與溫度**
- **品牌或廠商類型**（煨小寶 / 津沅寶 / 呷8味 / 代工）
- **份量設計**（每份重量、本包裝含幾份）

若缺少上述任何項目，先詢問使用者補充，再產出標示。

> **字型**：所有 Word 輸出一律使用 **Noto Sans TC**，傳入 `generate_docx.py` 時無需另行指定。

> **素食標示**：此欄位已從標示格式移除，**不再輸出素食標示欄位**。

## 廠商資訊處理規則（重要）

### 自有品牌（煨小寶 / 津沅寶 / 呷8味）
自動從 `references/brand-info.md` 帶入立康食品科技股份有限公司的完整資訊。

### 代工產品（OEM）
台灣食品標示實務上，代工產品通常採**品牌商負責制**：標示欄位填入**客戶（委託方）的公司資訊**，立康食品為實際製造廠但不一定出現在包裝正面標示。

**欄位標籤變更**：OEM 產品的廠商欄位抬頭由「製造廠商」改為「**負責廠商**」。
在 input JSON 中設定 `"is_oem": true` 即可自動切換。

**必須向使用者詢問以下客戶資料：**
1. 委託廠商（客戶）公司名稱
2. 客戶電話
3. 客戶地址
4. 工廠登記號碼（客戶的，或依合約使用立康的 93773108）

建議詢問話術：「這是代工產品，標示上的負責廠商要填客戶的資料，請提供：客戶公司名稱、電話、地址、工廠登記號碼。」

若使用者未提供，先以「（待填）」佔位並在備註提醒補充。

若使用者未提供營養成分數值，主動告知「將依台灣食品營養成分資料庫估算，建議以原料規格書最終校正」，並進行估算後填入。

## 標示產出流程

### Step 0｜查詢商用原料資料庫

在處理成分表之前，先掃描配方中是否有**商用複合原料**（如雞粉、辣椒醬、醬油、椰漿等加工食品原料）。

對每個商用原料，**依序**查詢以下兩個路徑：

**路徑一（優先）：立康原料規格圖庫**
```
G:\我的雲端硬碟\立康-產品&原物料成本\原料成分&營養標
```
- 用 `Glob` 搜尋品名關鍵字（如：`*金蘭*醬油*`、`*雞粉*`）
- 圖庫內容為 JPG / PNG / AVIF / webp 格式的原料包裝照片
- **找到圖片** → 用 `Read` 工具直接讀取圖片（Claude 具備圖像辨識能力），從中萃取：
  - 成分表（子原料依序列出）
  - 每100公克營養成分（熱量、蛋白質、脂肪、飽和脂肪、反式脂肪、碳水化合物、糖、鈉）
  - 過敏原聲明
- 若同一原料有多張圖（成分 + 營養標示分開），全部讀取後合併資訊

**路徑二（備用）：結構化 Markdown 規格書**
```
C:\Users\hchen\.claude\skills\food-label\references\ingredients
```
- 適用於已手動建檔的原料（格式參見 `references/ingredients/README.md`）
- **找到 .md 檔** → 讀取「子成分」與「每100公克營養成分」

**兩個路徑都找不到時：**
- 改用 TFND 查詢（`python scripts/lookup_nutrition.py <原料名稱>`）估算
- 告知使用者：「{原料名稱} 規格書尚未建檔，已依 TFND 資料庫估算，建議後續將規格圖片存入 G:\我的雲端硬碟\立康-產品&原物料成本\原料成分&營養標 以提升準確度」

> **搜尋技巧**：品名可用部分關鍵字比對，例如搜尋「醬油」可找到「金蘭珍味醬油」；搜尋「椰漿」可找到「佳樂Kara 椰漿」。

### Step 1｜成分表排序與格式化

1. 將所有原料依**添加量由多到少**排序。
2. 複合原料（如醬油、沙拉醬）需在原料名稱後以括號展開子原料，同樣依比例排序。
3. 添加物（食品添加物）須標示功能名稱（如「防腐劑」）與許可名稱（如「苯甲酸鈉」）。
4. 基改原料：若含黃豆、玉米等高風險作物，加註「非基因改造」或「基因改造」。
5. 輸出格式：以頓號「、」分隔，複合原料用方括號「[ ]」或圓括號「( )」展開。

### Step 2｜過敏原偵測

讀取 `references/allergen-rules.md`，自動比對成分表，產出過敏原警語。
格式：「本產品含有 ○○、○○ 及其製品。」

### Step 3｜素食判定與品名標註

**素食類型判定：**
- 含肉類、魚類、禽類、海鮮、動物內臟 → **非素食**（品名不加任何括號）
- 完全植物性，無五辛 → **純素**
- 含蛋不含奶（且無肉類）→ **蛋素**
- 含奶不含蛋（且無肉類）→ **奶素**
- 含蛋奶（且無肉類）→ **蛋奶素**
- 完全植物性但含蔥蒜韭薤興蕖（五辛）→ **植物五辛素**

**品名標示規則（台灣素食標示規範）：**
- 素食產品（含任何素食類型）→ 品名後括號加註素食類型
  - 格式：`{品名}（{素食類型}）`
  - 例：`麻油猴頭菇湯（純素）`、`素燥飯（蛋奶素）`
- 非素食產品 → 品名不加任何括號
  - 例：`燉雞湯`、`紅燒豬腳`

此標註出現在：品名欄位、Word 文件標題、成分表品名列。
**檔名仍使用純品名**（不含素食括號），避免特殊字元造成問題。

在 input JSON 中傳入 `vegetarian_type` 欄位：
- 素食：填 `"純素"`、`"蛋素"`、`"奶素"`、`"蛋奶素"`、`"植物五辛素"`
- 非素食：留空 `""`或省略

### Step 4｜廠商資訊填入

讀取 `references/brand-info.md` 取得對應品牌的廠商資訊。
代工產品則由使用者提供，或先填「（待填）」。

### Step 5｜營養標示格式化

依法定格式輸出兩欄：每份 / 每100公克（或每100毫升）。

必填項目（依序）：
1. 熱量（大卡）
2. 蛋白質（公克）
3. 脂肪（公克）→ 展開：飽和脂肪、反式脂肪
4. 碳水化合物（公克）→ 展開：糖
5. 鈉（毫克）

若產品宣稱功能性（如高鈣、高纖），需額外標示相關營養素。

#### 營養成分資料來源優先順序

1. **使用者直接提供數值** → 最優先，原樣使用
2. **商用原料規格書**（`references/ingredients/` 中有紀錄的）→ 直接採用規格書數值
3. **TFND 台灣食品成分資料庫（2025版）** → 呼叫查詢腳本：
   ```bash
   python scripts/lookup_nutrition.py <食材名稱> --top 3
   ```
   - 從結果中選擇最符合的品項（優先選同食品分類、相近製備狀態）
   - 依各食材用量加權計算整體每100g數值
   - 若找不到特定食材，告知使用者並使用相近食材估算

**加權計算方式（每100g成品）：**
```
各成分每100g數值 × (該成分用量g / 100) → 加總 → ÷ 成品總重量g × 100
```
例：成品500g中有雞腿肉200g（熱量165大卡/100g）→ 熱量貢獻 = 165 × 2 = 330大卡（為500g成品的66大卡/100g）

**反式脂肪注意**：TFND 中反式脂肪欄位為 mg，腳本已自動換算為 g，無需手動轉換。

### Step 6｜完整標示草稿

組合所有欄位，依下列順序輸出完整標示文字（**不含素食標示欄位**）：

```
品名：○○○
成分：○○、○○（…）、○○…
淨重：○○公克 ±○○公克（固形物：○○公克）
有效日期：標示於包裝上
保存方式：請於○○保存
原產地：台灣
過敏原：本產品含有○○及其製品。
製造廠商：○○食品科技股份有限公司
電話：○○
地址：○○
工廠登記號碼：○○
營養標示：
  每份量：○○公克　本包裝含：○份
  [表格]
```

### Step 7｜產出 Word 文件

執行 `scripts/generate_docx.py`，輸出格式化 .docx。

**檔名格式（固定）**：`產品標示_{品名}_{YYYYMMDD}.docx`
- 例：`產品標示_燉雞湯_20260628.docx`
- 品名取純品名（不含素食括號）
- 日期取產出當天日期

呼叫方式（腳本自動產生檔名）：
```bash
python scripts/generate_docx.py input.json
```
**預設輸出路徑**：
```
C:\Users\hchen\Desktop\SynologyDrive\03【存】產品 (照片, 條碼, 檢驗報告, 目錄)
```
若需輸出到其他位置，可指定第二個參數覆蓋：
```bash
python scripts/generate_docx.py input.json "其他資料夾路徑"
```

## 注意事項

- 所有單位使用中文法定單位（公克、毫克、大卡、毫升）
- 有效日期：若未知確切標示位置，填「標示於包裝上」
- 原產地：除非使用者另指定，預設填「台灣」
- 若使用者提問涉及法規查詢（如添加物限量、宣稱規範），直接回答，不需等待標示輸出

## 淨重容許負誤差計算規則

依《定量包裝商品管理辦法》附表，**不得**使用固定 ±5%，須依標示量（N）查表計算：

| 標示量 N（公克或公撮） | 容許負誤差 t |
|----------------------|-------------|
| 5 – 50               | 9.0%        |
| 50 – 100             | 4.5公克      |
| 100 – 200            | 4.5%        |
| 200 – 300            | 9公克        |
| 300 – 500            | 3.0%        |
| 500 – 1000           | 15公克       |
| 1000 – 10000         | 1.5%        |
| 10000 – 15000        | 150公克      |

在 input JSON 傳入 `nominal_weight_g`（純數字），腳本會自動計算並格式化。

**常見範例：**
- 750公克 → 500–1000 區間 → **±15公克**
- 500公克 → 300–500 區間 → **±15公克（3%）**
- 800公克 → 500–1000 區間 → **±15公克**
- 150公克 → 100–200 區間 → **±6.75公克（4.5%）**
